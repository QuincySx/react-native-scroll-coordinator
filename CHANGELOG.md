# Changelog

All notable changes are documented here. The project follows Semantic
Versioning once it reaches `1.0.0`; prerelease versions may still refine the
public API.

## 0.1.0-alpha.0

- Added Android-native collapsible header, sticky tabs, pager coordination,
  continuous nested fling, and horizontal edge arbitration.
- Added FlatList, optional FlashList, custom virtual-list, iOS, and Web
  adapters.
- Added ESM, CommonJS, Codegen, release-contract, and example applications.

This alpha is production-evaluation software. See
`docs/COMPATIBILITY.md` for verified and experimental platform support.

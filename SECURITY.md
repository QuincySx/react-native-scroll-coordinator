# Security policy

## Supported versions

Only the newest published prerelease is supported before `1.0.0`. After
`1.0.0`, the latest minor release of the current major receives fixes.

## Reporting

Do not open public issues for vulnerabilities. Use the repository's private
security advisory flow and include affected versions, reproduction steps,
impact, and any proposed mitigation:
https://github.com/QuincySx/react-native-scroll-coordinator/security/advisories/new

This library does not process credentials or user data, but native crashes,
dependency substitution, malicious package publication, and arbitrary code
execution are security issues.
